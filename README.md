# Cinema – Movie List & Cost Tracker (HTML/CSS/JS)

A small app that lists movies with images, lets you “watch” (toggle) them, and tracks the total cost in RSD.

## Demo
![Demo](bioskop/gif_cinema.gif)

## Features
- Render movies (image, title, description, rating, price)
- “Watch / Cancel” toggle per movie
- Live total cost update
- Visual state for watched movies

## Built With
- HTML  
- CSS  
- Vanilla JavaScript
